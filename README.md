# test-design-dify
## 1、dify是什么

本项目使用 Dify构建，Dify 是一个开源的 LLM 应用开发平台，提供可视化的工作流编排和 RAG 引擎，让 AI 应用开发更加简单高效。

项目地址：[langgenius/dify: Production-ready platform for agentic workflow development.](https://github.com/langgenius/dify)

推荐使用cloud dify快速体验工作流的编排：[工作室 - Dify](https://cloud.dify.ai/apps)

## 2、项目介绍

通过学习该项目你可以学习到简单的工作流编排方式、如何攥写prompt提示词，如果你正好需要使用大模型依据需求文档生成测试设计，该项目可以直接应用。
